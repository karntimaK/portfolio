Rental Comic Book System

UX/UI Case Study · Web Application

บทบาท: UX/UI Designer · เครื่องมือ: Figma

ภาพรวมโปรเจค

ระบบเช่าหนังสือการ์ตูน (Rental Comic Book System) เป็นเว็บแอปพลิเคชัน ที่ออกแบบมาเพื่อช่วยให้ผู้ใช้สามารถค้นหา ดูรายละเอียด และเช่าการ์ตูน ได้อย่างสะดวกและเข้าใจง่าย

ปัญหาที่พบ

  • ผู้ใช้ไม่ทราบว่าการ์ตูนเล่มใดว่างให้เช่า
  • ขั้นตอนการเช่าซับซ้อนสำหรับผู้ใช้ใหม่
  • สถานะการยืมคืนไม่ชัดเจน

เป้าหมายในการออกแบบ

  • ทำให้ขั้นตอนการเช่าการ์ตูนเข้าใจง่าย
  • แสดงสถานะการเช่าอย่างชัดเจน
  • เพิ่มประสบการณ์การใช้งานที่ดีให้กับผู้ใช้

User Persona

ชื่อ: เคน (อายุ 21 ปี)

เป้าหมาย: เช่าการ์ตูนได้อย่างรวดเร็ว

ปัญหา: ระบบใช้งานยากและไม่ชัดเจน

User Flow

เลือกการ์ตูน → ดูรายละเอียด → กดเช่า → ยืนยันการเช่า

Wireframe

ออกแบบ Wireframe แบบ Low-fidelity เพื่อโฟกัสที่โครงสร้างหน้า และลำดับการใช้งานของผู้ใช้

UI Mockup

การออกแบบ UI เน้นความอ่านง่าย ใช้รูปแบบการ์ด และแสดงสถานะการเช่าอย่างชัดเจน

เหตุผลในการออกแบบ

  • ใช้ Card Layout เพื่อให้ผู้ใช้สแกนข้อมูลได้ง่าย
  • ใช้ป้ายสถานะเพื่อบอกความพร้อมในการเช่า

Figma Prototype

สามารถดู Prototype และ Flow การใช้งานของโปรเจคนี้ได้ผ่าน Figma โดยลิงก์ด้านล่าง
