Rental Comic Book System

UX/UI Case Study · Web Application

Project Overview

This project focuses on designing a rental comic book system that allows users to browse, rent, and manage comic books easily.

Problem

  • Comic rental systems are often confusing
  • Users struggle to find available comics
  • Rental status is unclear

Solution

Design a clear system with simple navigation, visible rental status, and an easy rental process.

User Persona

Name: Ken (21)
Goal: Rent comics quickly without confusion
Pain Point: Doesn’t know which comics are available

User Flow

  1. Browse comic list
  2. View comic details
  3. Check availability
  4. Confirm rental
  5. View rental history

UX Decisions

  • Use clear status labels (Available / Rented)
  • Limit steps in rental flow
  • Use simple card-based layout
