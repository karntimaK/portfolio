Nongwaeng Salon Appointment Booking

UX/UI Case Study · Mobile Application

บทบาท: UX/UI Designer · เครื่องมือ: Figma

ภาพรวมโปรเจค

Nongwaeng Salon เป็นแอปพลิเคชันสำหรับจองคิวร้านทำผม ที่ออกแบบมาเพื่อแก้ปัญหาการจองผ่านแชท และช่วยให้ผู้ใช้เลือกวันเวลาได้อย่างสะดวก

ปัญหาที่พบ

  • การจองคิวผ่านแชทใช้เวลานาน
  • ผู้ใช้ไม่เห็นช่วงเวลาที่ว่างชัดเจน
  • การยืนยันการจองไม่สะดวก

เป้าหมายในการออกแบบ

  • ลดขั้นตอนการจองคิว
  • แสดงช่วงเวลาที่ว่างอย่างชัดเจน
  • เพิ่มความมั่นใจให้ผู้ใช้ก่อนยืนยันการจอง

User Persona

ชื่อ: เมย์ (อายุ 25 ปี)

เป้าหมาย: จองคิวร้านทำผมได้อย่างรวดเร็ว

ปัญหา: ระบบจองยุ่งยากและเสียเวลา

User Flow

เลือกบริการ → เลือกช่าง → เลือกวันและเวลา → ยืนยันการจอง

Wireframe

Wireframe ถูกออกแบบเพื่อเน้นการใช้งานบนมือถือ และลดขั้นตอนที่ไม่จำเป็น

UI Mockup

ใช้โทนสีอ่อน ปุ่มขนาดใหญ่ เพื่อให้ใช้งานง่ายและเป็นมิตรกับผู้ใช้

เหตุผลในการออกแบบ

  • ใช้ปุ่ม CTA ขนาดใหญ่เพื่อรองรับการใช้งานบนมือถือ
  • ออกแบบ Flow แบบ Step-by-step ลดความสับสน

Figma Prototype
