Nongwaeng Salon

Appointment Booking · Mobile Application

Project Overview

Nongwaeng Salon is a mobile application designed to help customers book salon appointments easily without calling or waiting.

Problem

  • Customers need to call to make appointments
  • Long waiting time
  • No clear schedule overview

Solution

Design a mobile booking system that allows users to select services, stylists, and time slots easily.

User Persona

Name: Nook (26)
Goal: Book a salon appointment quickly
Pain Point: No time to call the salon

User Flow

  1. Select service
  2. Choose stylist
  3. Select date & time
  4. Confirm appointment

UX Decisions

  • Show available time slots only
  • Use step-by-step booking flow
  • Clear confirmation screen

UI Concept

  • Soft colors (friendly & relaxing)
  • Large buttons for mobile use
  • Readable typography
